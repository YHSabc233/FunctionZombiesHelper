# Function Zombies Helper  
[English](README.md) | 简体中文(有更新日志)

一个为**函数僵尸末日**制作的辅助模组。

## 功能

- 血量显示
- Show Spawn Time
- 也许还有更多？emm...

请前往 [Modrinth](https://modrinth.com/mod/function-zombies-helper) 获取模组文件。
